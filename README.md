# Enterprise-Sales-Operations-MIS-

##  Project Overview
Complete MIS reporting system built for 
Nexus Retail Solutions, analyzing 9,994 
sales transactions across 4 US regions.

##  Tools Used
- Advanced Excel (SUMIFS, INDEX-MATCH, 
  COUNTIFS, nested IFS)
- Power Query (ETL automation)
- Power Pivot (Data Modeling)
- VBA (Report Automation)
- Pivot Tables & Charts
- Interactive Slicers & Timeline

- ##  Key Features
- 20 KPIs with RAG status tracking
- Power Query automated data refresh
- VBA macro for one-click report generation
- Interactive dashboard with cross-filtering
- Regional drill-down analysis
- Professional documentation

- ##  Key Findings
- Technology is highest revenue category
- West region leads in total sales
- Consumer segment = largest revenue share
- High discount orders show profit erosion

- ##  Project Structure
| Sheet | Purpose |
|-------|---------|
| RAW_DATA | Original source data |
| DATA_DICTIONARY | Column documentation |
| CLEANED_DATA | Power Query output |
| ANALYSIS_ENGINE | Formula calculations |
| KPI_TRACKER | 20 KPIs with RAG status |
| EXECUTIVE_DASHBOARD | Interactive visual dashboard |
| REGIONAL_REPORT | Geographic drill-down |
| AUTOMATED_REPORT | VBA-generated monthly report |

# Enterprise Sales & Operations 
# MIS Reporting System

## Overview
Complete MIS reporting system built 
for Nexus Retail Solutions, analyzing 
9,994 retail transactions across 
4 US regions and 3 product categories.

Designed to replicate real MIS Executive 
work automated, documented, and 
stakeholder-ready.

##  Tools and Technologies
| Tool | Purpose |
|------|---------|
| Advanced Excel | Formulas, KPI calculations |
| Power Query | Automated ETL pipeline |
| Power Pivot | Data model and relationships |
| VBA | Monthly report automation |
| Pivot Tables | Dynamic data summarization |
| Dashboard Design | Executive visualization |

## Key Features
- 20 KPIs with RAG status tracking
- Power Query automated data refresh
- Power Pivot 3-table data model
- VBA macro — one-click report generation
- Interactive dashboard with slicers
- Regional drill-down with dropdown
- Professional documentation

##  Project Structure
| Sheet | Purpose |
|-------|---------|
| RAW_DATA | Original source — never edited |
| DATA_DICTIONARY | Column documentation |
| README | User guide |
| ASSUMPTIONS | Business rules documented |
| CLEANED_DATA | Power Query output |
| ANALYSIS_ENGINE | Formula calculations |
| KPI_TRACKER | 20 KPIs with RAG status |
| EXECUTIVE_DASHBOARD | Interactive dashboard |
| REGIONAL_REPORT | Geographic analysis |
| AUTOMATED_REPORT | VBA-generated report |

## Key Business Findings
- Technology generates highest revenue
- West region leads in total sales
- Consumer segment = 50%+ of revenue
- High discount orders erode profit margin
